FurrEver : A Digital Platform for AI Pet Matching and Report Abuse.

Features : 

    Adopt Pet: Browse and adopt pets easily from shelters and individuals.
  AI-based Pet Matching: Recommends pets based on preferences,lifestyle, and compatibility.
  
  Abuse Reporting System: Report and track animal abuse cases to protect animal welfare.
  
  Chatbot: Provides guidance and answers queries for pet adopters.
  
  Grooming Services: Information and guidance about pet grooming.
  
  Community Hub: Connect with fellow pet owners, share experiences,and participate in discussions.
  
  Health Care Services: Locate nearby veterinary clinics and pet
  pharmacies for medical support.
  
  Breed Detection: Allows users to upload an image of a pet and
automatically identifies its breed using image recognition techniques.

Technology Stack : 

  Frontend : HTML , CSS , JavaScript

  Backend : Python Flask

  Data Base : MYSQL

  Machine Learning: Random Forest Model for pet recommendation
and TensorFlow model for pet image recognition
